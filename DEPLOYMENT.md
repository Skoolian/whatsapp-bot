# VPS Server-এ হোস্ট করার গাইডলাইন

আপনার WhatsApp বটটি VPS সার্ভারে হোস্ট করার জন্য নিচের ধাপগুলো অনুসরণ করুন।

## ১. সার্ভার প্রস্তুতি (Prerequisites)

আপনার সার্ভারে **Node.js (v18 বা তার বেশি)** এবং **PM2** ইন্সটল থাকতে হবে।

```bash
# Node.js ইন্সটল করার কমান্ড (Ubuntu/Debian)
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs

# PM2 গ্লোবাললি ইন্সটল করুন
sudo npm install -g pm2
```

## ২. প্রোজেক্ট সেটআপ

প্রথমে আপনার প্রোজেক্ট ফাইলগুলো সার্ভারে আপলোড করুন অথবা Git দিয়ে ক্লোন করুন। তারপর প্রোজেক্ট ডিরেক্টরিতে গিয়ে ডিপেন্ডেন্সিগুলো ইন্সটল করুন:

```bash
cd new-bot-2026
npm install
npm rebuild better-sqlite3
```

## ৩. পোর্ট কনফিগারেশন

বটটি এখন **3051** পোর্টে কনফিগার করা হয়েছে। আপনি যদি অন্য কোনো পোর্টে চালাতে চান, তবে `bot.js` ফাইলের একদম নিচে `PORT` পরিবর্তন করতে পারেন অথবা এনভায়রনমেন্ট ভেরিয়েবল ব্যবহার করতে পারেন:

```bash
export PORT=8080
```

## ৪. PM2 দিয়ে রান করা

বটটি ২৪ ঘণ্টা চালু রাখার জন্য এবং সার্ভার রিস্টার্ট হলেও যেন অটোমেটিক চালু হয় সেজন্য PM2 ব্যবহার করুন:

```bash
# বট চালু করুন
pm2 start bot.js --name "whatsapp-bot"

# রিস্টার্ট হলেও যেন অটো চালু হয়
pm2 save
pm2 startup
```

## ৫. চেক করা

বটটি ঠিকমতো চলছে কিনা তা দেখতে:
```bash
pm2 logs whatsapp-bot
```

## ৬. গুরুত্বপূর্ণ টিপস
- সার্ভারের ফায়ারওয়ালে (Firewall) আপনার কাঙ্ক্ষিত পোর্টটি (যেমন: 3051) ওপেন করে দিন।
- যদি `Error: globalThis.crypto is undefined` আসে, তবে নিশ্চিত করুন যে আপনি আমার তৈরি করা `system.js` এবং আপডেটেড `bot.js` ব্যবহার করছেন।
